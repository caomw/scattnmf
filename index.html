Scattnmf

Code implementing the model from "Source Separation with Scattering NMF", by J. Bruna, P. Sprechmann and Y. LeCun, ICASSP 2015. (http://cims.nyu.edu/~bruna/Publications_files/paper_icassp.pdf)

Run

To see the code in action, run demo.m

Notes

  • Training of dictionaries requires the matlab package SPAMS (http://spams-devel.gforge.inria.fr).

  • Some parts of the scattering code are borrowed from the Scatnet package (https://github.com/scatnet/scatnet)
